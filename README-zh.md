# Yolov5ForCSGO

## 项目介绍

本项目利用 [yolov5](https://github.com/ultralytics/yolov5) 实现 CSGO (Counter-Strike:Global Offensive)游戏中的人物检测

## 快速上手(使用教程)

1. 克隆本项目

	git clone git@github.com:Lucid1ty/Yolov5ForCSGO.git

2. 切换到本项目的目录下

	cd Yolov5ForCSGO

3. 安装环境依赖

	pip install -r requirements.txt
	
4. 使用
	施工中...
